7.อธิบายการทำงาน ในรูปแบบ MVC มาพอสังเขป
ตอบ MVC MODEL, VIEW, CONTROLLER เป็น Pattern ใช้หลักการ OOP แบ่งการทำงานอย่างชัดเจน ทำให้ง่ายต่อการทำงาน และจัดการ
โดย M จะจัดการส่วนของข้อมูลเก็บเป็น object 
    V เป็น่สวนหน้าตาของโปรแกรม ดึงข้อมูลมาแสดงให้ผู้ใช้เห็น หรือก้คือส่วน GUI
    C ทำหน้าที่รับ และจัดการทำงานในส่วนที่เป็น logic ทั้งหมดในระบบ

8.อธิบายการทำงาน Git
ตอบ จะมี folder สำหรับเก็บไฟล์งานของเรา หลังจาก Tracked Git แล้ว จะเริ่มการตรวจสอบไฟล์ต่างๆ ที่ถูกเพิ่มเข้ามา หรือแก้ไข ภายใน folder
Staging Area การที่เราแก้ไข หรือเพิ่มไฟล์ สถานะของไฟล์จะเปลี่ยนเป็น Staging Area คือไฟล์ที่สามารถ Commit เพื่อเก็บประวัติการแก้ไขแต่ละครั้งได้
Local Repository ทุกครั้งที่เรา เพิ่ม หรือ แก้ไขไฟล์ และ Commit ข้อมูลจะถูกเก็บไว้ใน Local เพื่อเตรียม push
Remote Repository ใชช้คำสั่ง Push เพื่อนำไฟล์ข้อมูลทั้งหมด ขึ้นไปเก็บไฟล์บน Remote Server
